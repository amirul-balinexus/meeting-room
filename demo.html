Meeting Room Calendars - Native Version

🚀 Quick Start

This is a pure HTML/CSS/JavaScript implementation of the meeting room calendar application.

Open Application

✨ Features

  • Home Page: Grid layout of meeting rooms with hover effects
  • Room Pages: Full-screen Google Calendar integration
  • Responsive Design: Works on all device sizes
  • Client-side Routing: Browser back/forward support
  • Auto-refresh: Calendars refresh every 5 minutes
  • Fullscreen Mode: Dedicated fullscreen view
  • Direct URL Access: Navigate directly to room URLs
  • Dynamic Room Management: Add and update rooms programmatically

🏗️ Technical Details

  • No Build Process: Open HTML file directly in browser
  • No Dependencies: Pure HTML, CSS, and JavaScript
  • Modern CSS: Grid, Flexbox, and CSS animations
  • ES6+ JavaScript: Modern JavaScript features
  • Responsive Design: Mobile-first approach

📱 Browser Support

Works in all modern browsers:

  • Chrome 60+
  • Firefox 55+
  • Safari 12+
  • Edge 79+

🔧 How to Use

  1. Click "Open Application" above
  2. Select a meeting room from the home page
  3. View the calendar in full-screen mode
  4. Use the fullscreen button for dedicated view
  5. Navigate back using the back button or browser controls
